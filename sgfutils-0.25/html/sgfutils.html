sgfutils

The package sgfutils is a collection of command line utilities that help working with SGF files, especially when they describe go (igo, weiqi, baduk) games.

  • sgf - polish SGF file
  • sgfcharset - convert SGF file to UTF-8
  • sgfcheck - check SGF file
  • sgfcmp - compare two SGF files
  • sgfsplit - split SGF file (into games)
  • sgfvarsplit - split SGF file (into variations)
  • sgfstrip - strip SGF file
  • sgftf - transform SGF file
  • sgfmerge - merge SGF files
  • sgfinfo - list properties of selected games
  • sgfdb - build a game data base
  • sgfdbinfo - find games by pattern or properties
  • sgfx - extract data for a single game/problem
  • sgftopng - create diagrams
  • gib2sgf, ngf2sgf, ugi2sgf - convert GIB or NGF or UGF to SGF

Distribution

The package is provided as C source, written for a Linux environment.
